Use Ctrlp's g:ctrlp_switch_buffer instead of tab drop

tab drop allows you to switch to the tab with the specified name if it is already open. But it is very inefficiently done. It iterates through each tab until it finds the matching file. It is slow, and a custom function like 'switch to the last used tab' fails. ctrlp however has an option which fixes this. ctrp is a plugin that searches and opens recent files, files in the current directory, and files opened in the buffer.
