Copying code to Python REPL and avoid Indentation error

REPL stands for Read-eval-print loop for those who are uninitiated. It the prompt that shows up when you type python on your command line. You might innocuously copy code (like a function) from another website and get frustrated by garbled output with "Indentation error" as soon as you hit the 'paste' key or Shift+Insert. The solution is simple. Just get rid of the blank lines.

cd $TOP/../thirdparty && git reset --hard && git pull --rebase && git reset --hard $(grep TOOLCHAIN_GIT_HASH $TOP/Makefile.toolchain | head -n1 | cut -d'=' -f2) && make && make install && cd $TOP && make BUILD_TYPE=release -j4 CCACHE_DIR=~/.ccache
