Using git for hg repository.
October 17, 2014 [ dev ] #versioning, #git, #hg Read in 1 min.

I recently had to work with a project that is in mercurial, but I know that git is more powerful and is more widely used. So I looked around and there is a way to use git on hg repository.. Sorry if I am late to the party, but I am so happy. Because seriously, when there are two alternatives, I would like to stick to the best one. wink wink. If you must use mercurial. There is an excellent article written by Jamie Wong about setting up mercurial like pro, but make sure you check out lawrencium instead of Jamies' mercenary vim plugin, because Jamie moved on to git too! Nice move. :).

But seriously please keep using mercurial, git folks need motivation sometimes.

Here is the README for git-remote-hg. I do not know if hg-git is better, but git-remote-hg works fine for me. By the way, don't quote the URL like shown in the README page.

And don't forget running git gc --aggressive
