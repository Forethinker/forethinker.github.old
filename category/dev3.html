Here are some Bash commands that I used

Rename

Rename using find and sed

find . -name "*syntax.md" | sed -e "p;s/syntax/notes/" | xargs -n2 mv

Execute some commands based on the filename or extension 1

find . -name "*.txt" | xargs -I{} sh -c 'base=$(basename $0) ; name=${base%.*} ; ext=${base##*.} ; mv "$0" "foo/${name}.bar.${ext}"' {}

note: the variable name referenced by -I argument, which in this example is {} cannot be reference used for variable expansion. It must be assigned to another variable first.
