MongoDB Notes

October 12, 2013 [ dev ] #python Read in 4 min.

MongoDB is dynamically typed, meaning you can insert any data type as long as they are well-formated. Each object is associated with an ObjectId.

Cursor object is a Python dict once it has been assigned to a variable.

installation

sudo apt-get install ruby1.9.1-dev rubygems
sudo gem install require genghisapp bson_ext rake
sudo gem install bson_ext -v 1.9.2
genghisapp
genghisapp --kill; genghisapp

NOTE: You might run into the following error message:

Invalid gemspec in [/var/lib/gems/1.8/specifications/vegas-0.1.11.gemspec]: invalid date format in specification: "2009-08-30 00:00:00.000000000Z"

To workaround the above error, run the following cmd:

sudo sed -i 's/ 00:00:00.000000000Z//' /var/lib/gems/1.8/specifications/*

Attempt starting genghisapp again by running:

genghisapp

mongo is built-in client to the database

mongo localhost:27017
use DATABASE_NAME

Ctrl-D to exit out of the script

term:

database: database collections: tables documents: rows (objects)

The key may have alphanumeric names matching the document or a number matching the index of the element (starts from 0), each nest level is separated by ., so no names are allowed to have . in them. The value always match the value of the key referred by the element of the last nested level. The key must be wrapped in double quotes if any of dot operator is used.

, implies AND in query, OR must be queried explicitly using $or insert({}): can take an object or a list of objects and returns the ObjectIds assigned to each element as a list. find({}): takes in objects (comma delimited) as the filter for the search. It returns a Cursor object and must be type casted with list(). find_one({}): like find(), but returns the first result only. Does not need to be casted. update({},{},OPT): query, post, and options. upsert=True/False inserts instead of updating if the query is not found. manipulate=True/False SOManipulator will be applied before updating and the update will return the ID of the newly created document. Just choose False. Used to modify multiple documents save({}): both query and post information is stored in a single object separated by comma, modifies ONE document.

skip(#): skips the first # results. limit(#): limit the number of the result, return a list as a Cursor object so it must be casted even when the limiting number is 1. sort(field, OPT): OPT is either DESCENDING, OR ASCENDING (default).

ensure_index(''): takes a key (string) between quotes and creates an index on that key.

drop(): drop a collection remove({}): find and then delete. update({}): remove count()

$project

$inc Increment a numeric field (generalized; can increment by any number) $in find an element in the list that matches $set Set certain fields to new values $unset Remove a field from the document $push Append a value onto an array in the document $pushAll Append several values onto an array $addToSet Add a value to an array if and only if it does not already exist $pop Remove the last (or first) value of an array

backup and restore

mongodump --host <host_name>:<port> --db=<DB_name> --collection=<collection_name> --query '{}'
mongorestore --host localhost:27017 --db <DB_name> --collection=<collection_name> --drop <directory to drop to>

drop option clears the matching DB / collection before the restoration
