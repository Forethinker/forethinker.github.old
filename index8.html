useradd -m -g groupName -s /sbin/nologin userName

Add User to samba and create a new password:

pdbedit -a -u userName

Remove User from samba (modify the user by adding a new one):

Printing on Linux
November 18, 2013 [ hack ] Read in 1 min.

Both major printer manufacturers support Linux drivers: Canon and Brother. They can be downloaded from their support pages. Depending on the printer driver, you probably will need lpr, but definitely cups to keep your sanity intact.

sudo apt-get install lpr cups

cups works well with root and not other privileged account and if you do not have any password on the account, you need to create one.

sudo su
passwd

or simply sudo passwd root
