Going back to Windows as my primary OS

February 16, 2014 [ hack ] #rant Read in 2 min.

Windows is on a downhill since the new Windows Vista and I have considered to switch to Linux full time and I almost have, but now I am getting a bit sick of going after drivers to make everything run properly. Windows 7 still provides a great user experience (in terms of productivity, at least) out of the box.

Here are few things I miss dearly: 1. device/hardware support (wifi, trackpad): I managed to get them to work, but I spent like 30 hours.

  1. Everything Search: I have improved my find skill grep skill with Linux, but I miss Everything Search because I do no thave to think so hard with it.

  2. file manager: I am not asking for Directory Opus, but even something like Total Commander is not available. Double Commander was buggy and it was not that well documented.

  3. Microsoft Office: Office Software Suite is still dominated by Microsoft. I wish there was more competition. Libre Office, quietly frankly, resembles Office XP. And for a keyboard shortcut enthusiast like me, Google Doc is not in the same list for the comparison.

  4. Adobe Suite: Adobe is the Microsoft of Multimedia Editing, enough said.

  5. backup software: backing up hot disk and restoring them easily is not the forte of Linux. My choice on Windows is Paragon.

  6. Other software: Like Google Pinyin keyboard, TeamViewer (keyboard binding is messed up with OpenBox), Regex Helper, Tax Turbo, Spotify, and VMWare Viewer. Even Dropbox crapped out on UTF-8 file name encoding at one point on Linux.

  7. Bad compatibility with i386 and amd64 packages: maybe there is a way around it, but some uses x86 version of libssl0.9.8, and the other uses x64. Maybe there is a way around it, but I miss the days when the applications I install would just work.

I still can't live without Linux, but from now on I am going to try to anything that has GUI front-end on Windows. I could see the usefulness of Windows 8's hybrid approach, but I do not like it. I think either Windows 7 should be forked into a separate branch of OS release for super users or Windows 8's desktop mode need to be similar to that of Windows 7 with the native support for configuration to disable any tablet optimized features.
