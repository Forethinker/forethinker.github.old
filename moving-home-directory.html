Moving Home Directory

October 25, 2013 [ hack ] Read in 1 min.

The following commnads changes the username and moves the home directory over:

usermod -l newuser -d /home/newuser -m olduser

but if you are like me and have the config files symlinked, they will break. Here is the command that will relink the symlinks for files:

find /home/user -type l -lname '/home/oldname/*' -print0 | xargs -0 -I{} -sh 'ln -nsf "$(readlink $1 | sed -e "s/$2/$3/")" "$1"' {} oldname newname

They will not however, relink the directory symlinks. That is more complicated. So I make organize the source directories and make systematic change with the following:

ln -sf $HOME/Dropbox/linux/crunchbang/* $HOME/.config/
