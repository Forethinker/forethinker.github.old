Sed, not sad
November 20, 2013 [ dev ] #linux Read in 1 min.

The separator can be any character as long as they are consistent

sed 'ADDRESS'Command fileName sed /PATTERN/AddressCommand fileName

  • -i: makes changes on the file, suppresses output. A file extension .bak can be used to redirect othe output.
  • ;: is to separate command within quotes
  • -e: is to connect two separate commands in quotes.
  • -n: suppresses output, useful when using custom output commands
  • -f: read the sed script from a file
