Windows is the best OS for Personal Use
January 25, 2014 [ hack ] #windows Read in 6 min.

"Computer User Survery"

When I was a senior in college, I asked all of my professors why they chose to use Mac (except one professor who was a respected contributor in Linux world and was a Ubuntu fan). Their reasons were convenience and reliability for the most part. Well, I find my time pretty valuable too, but I am sticking with Windows and here are Why. I am biased in my article, because I never actually owned a mac, just used them from time to time.

Operating System

  • Accessibility: Window's High Contrast Mode is quite good. I like it more than Apple's inversed screen, because all icons still remain in the same color. It can be quite confusing to my brain if things do not line up correctly with its memory. Windows keep things constant with their color scheme. I use Life Dark Theme.
  • Compatibility: I do not use any DOS program, but it is certainly good to know that everything will still work in the future.

Hardware

  • Matte Screen: glossy screens strain my eyes. Here Apple has taken the priority on aesthetic bit too far.
  • TPM: hardware encryption is overlooked by many people, but it is certainly a nice option considering their your data is worth more than any hardware or software you own.
  • Compatibility: hey, I do not have to check for MAC-compatible sign whenever I buy a new accessory for my machine! That is one less headache for me.

Software

These are the few of my favorite things - Autohotkey: ~~I have spent an enormous amount of time hunting for some good snippets of scripts online. This probably will save my health in the long run from RSI. And I am sure I am going to get back the time I invested in back before this program becomes obsolete (to me). I cannot emphasize enough its usefulness. And no, AppleScript is not equivalent to Autohotkey. I use spell check and auto completions everywhere.~~ I found that there is Autohotkey Equivalent on OS X called Sikuli - Chocolately: finally a software manager that is very very good. Their repository covers about 80% of the software I use. - Directory Opus: Path Finder 6 for Mac is looking quite good, but really Dopus is still the king of all file managers. - Microsoft Office: whether you hate it or not, it is the most popular office suite. Mac and Linux (via Wine) do not work the same way as the original version in Windows. For example, format preservation when copying does not work. It is something I would get with Mac anyway. - Everything: a very popular search program that is available exclusive on Windows. It makes me wonder if I need to learn to use grep properly. - JVM: Oracle optimized its start time for Windows, Linux, and Solaris, but not for OS X.

(Supposedly) The Shortcomings of Windows:

  • Security (Virus): I never had a problem with a virus since a decade ago when I started using an Anti-Virus program. I have always used my common sense.
  • Reliability: people who bash Windows for BSOD either have not used Windows lately, they were doing something crazy, used unsupported hardware, or were just unlucky. Using tools like Process Monitor helps you to identify the problem fairly quickly.
  • Maintenance: With some batch files, your maintenance could be a breeze. Add Chocolately and CCleaner to the bunch and you are all set. Yes deleting a software compeltely could take some time with Revo Uninstaller, but I have not had to reinstall Windows 7 since I have installed it.
  • Terminal: So far Cygwin has not failed me yet. Yes, it takes a ton of time to get things to work properly, but I am getting better at straighten things out. In a way it has been a good exercise for me because in the past, I had the attitude of "Never debug a program that I didn't make" - which was obviously a wrong attitude. Now that Windows 10 supports bash natively, all the better.
  • Driver Updates: this one I have to hand it over to Apple. This is more of an annual chore for me, because it must be done manually and I forget do it unless something is broken.

Linux, other thoughts, and conclusion

I have seen many of my peers who use Mac installing Parallel so that they could stay connected with the rest of the world. Maybe it is not the same thing, but I had a similar experience when I was using Linux as my primary OS. I just want to say that Windows is more prevalent than you think.

Here is another problem. The moment you use an Apple product, you are deemed as a source of revenues by technology marketers. You have to pay premium for everything! Hardwares, apps, and even accessories! Kudos to Apple for simplifying and enhancing the purchase experience, but I prefer keep my bank statement as lean as possible by using free software on Windows (and Android). Free PC hardwares are easy to come by as well.

I do not hate Apple or Mac. I love the fact that they are around to keep Microsoft on their toes. Just from personal experience, I found Microsoft to be more productive, that is all.

Using Windows in my opinion is better than Mac. I still glance over Mac's software offering from time to time, but it is not good enough to convince me over. At very minimum, it will have to be either Cygwin community shrinking or Autohotkey being supported on Mac before I switch over. I do not see either happening any time soon. I'd like to thank Cygwin community who has kept things going, asking for Windows support all over Github, and answering questions on Stackoverflow and Super User forums.

P.S. Windows 8

Windows 8's High Contrast Mode did not work the same way as Windows 7's when I tested it, but I am hoping it will change soon. Latest Dell Latitude series E6540 (w/ Intel Haswell) was surprisingly offered with Windows 7, denoting that Enterprise world is not adapting Windows 8. Likewise I would not use their tiled interface. I never used the start button to begin with. But once again, I am sold on Windows because of the things that go with it. So unless Microsoft really screws up and people abandon Windows, I probably will not need a sequel to this article.
