Windows Quirks
October 29, 2013 [ hack ] #windows, #complaints Read in 2 min.

Window's is nice, but it is not perfect. And when I Googled to find a way to file a bug report, I read someone saying that there isn't one for consumers, because more often consumers cannot distinguish OS bugs from their own mistakes/problems. Anyway, Here are my problems that I encountered and haven't found a solution to.

Windows 7

  • Ctrl, Shift, Win keys become sticky sometimes when I have completely disabled them.
  • The cmd program steals the focus of the window and there is nothing I can do about it.

Windows 8

  • There is no option to turn off the dark background like in Windows 7 (via Ease of Access button after C-A-Del).
  • I couldn't block websites with hosts file. I had to modify the DNS entry and add 127.0.0.1
  • The priority of the network devices were not being set even when changed on Control Panel -> Network -> Network and Sharing Center -> Change adapter settings -> Advanced -> Advanced Settings The following fixed it. Control Panel > All Control Panel Items > Network and Sharing Center > Change adapter settings (in the left pane), right-click a network, click Properties > highlight, in turn, each of the TCP/IP settings > Properties > General Tab > Advanced (at the bottom) and click the Automatic Metric box to clear the tick, then enter the value that you want in the Interface Metric field. The system defaults to the lower or lowest number, so, if you set Metric to 5 in the Wired Network connection and to 10 in the Wireless Connection, the system would default to the wired connection.
