Converting and Transferring Various Media Files
July 20, 2014 [ hack ] #media, #mp4, #audio, #video Read in 3 min.

Join mp4 videos in minutes with MP4Joiner

Mp4Joiner is the best program for joining two MP4 files together. Most of the camcorders and cell phones have file size limited to 4GB which translates to approximately 33 minutes for 1080p videos. If you try to merge two videos together with Sony Vegas or Adobe Premiere Pro, they re-render the entire video, taking more than four hours(!). But this program merely slice the metadata and merge the video together, taking less than 10 minutes. You can do the same thing with ffmpeg, but I like the optino of dragging and seeing the metadata on the application.

Splitting mp4 videos

This time I need to use ffmpeg. The vcodec and acodec options let me split the video file without re-encoding them, so that the operation will be done in minutes. ffmpeg.exe -i %userprofile%\Desktop\20150213.mp4 -vcodec copy -acodec copy -ss 00:00:00 -t 00:37:21 cut.mp4

Converting mp4 to mp3 with ffmpeg

ffmpeg is an amazing open source tool for any type of media conversion, but be aware that not all ffmpegs are compiled the same. I am using Windows Command Prompt.

"C:\ffmpeg\bin\ffmpeg.exe" -i C:\Users\john\Desktop\20140720.mp4 -vn -b:a 128k -c:a libmp3lame 20140720.mp3

Rotating mp4 180 degree

Converting pdf to images

sudo apt-get install -y poppler-utils
# create #150dpi resolution
pdftoppm -png file.pdf prefix
# create #300dpi resolution
pdftoppm -rx 300 -ry 300 -png file.pdf prefix

find . -name "*.pdf" -exec bash -c 'cd $(dirname "{}") && pdftoppm -png $(basename "{}") $(basename "{}" .pdf)' \;

The use of double quote around braces is very important, as it will defer the brace expansion later after the find has completed. Using single quotes will expand the braces before the find executes and return current directory.

Converting ppt(x) to pdf

Unconv cannot convert ppt(x) to png directly. It can only process the first page, which is a pretty lame behavior. But in conjunction with pdftoppm or convert (from imagemagik), the job can be done.

sudo apt-get install unoconv
unoconv --export Quality=100 filename.pptx filename.pdf
