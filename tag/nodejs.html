Grunt makes me smile
February 14, 2015 [ web ] #grunt, #nodejs Read in 1 min.

Some setup

sudo iptables -A INPUT -p tcp --dport 9000 -j ACCEPT Edit Grunt.js's host from localhost to 0.0.0.0 to access it from outside

Useful packages

They are pretty much all useful. These are the settings I put in grunt.initConfig({})

wiredep

First add these meta tags to index.html

<head>
    <!-- bower:css -->
    <!-- endbower -->
    <!-- bower:js -->
    <!-- endbower -->
</head>

in gruntFile.js:

grunt.loadNpmTasks('grunt-wiredep');

then in grunt.initConfig:

wiredep: {
  target: {
    src: 'index.html' // point to your HTML file.
  }
}

and execute it by:

grunt wiredep
