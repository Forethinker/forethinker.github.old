find ~/Dropbox/dev/dotfiles/ -maxdepth 1 -type f -name '.??*' -print0 | xargs -0I{} sh -c 'base="$(basename $0)"; if [ -f $HOME/"$base" ]; then  mv $HOME/"$base" $HOME/"$base".bak; echo "$0" {}; fi; ln -s "$0" $HOME/"${base}"' {}

Power Save

Follow the instruction from [here](http://okomestudio.net/biboroku/?p=1915#comment-14648)

Turn off the discrete graphics card

This will save a lot of battery. How much? Almost 30% of your battery capacity depending on your graphics card.

sudo su
apt-get install dkms
git clone https://github.com/Bumblebee-Project/bbswitch.git
cd bbswitch
# compile and install the module
make -f Makefile.dkms
#unload the nouveau driver - has accesses discrete graphics card and prevents from turning it off
modprobe -r nouveau
# activate the module
modprobe bbswitch

# to uninstall the module
make -f Makefile.dmks uninstall
# check the status of the NVIDA graphics card
cat /proc/acpi/bbswitch
# to turn it on
tee /proc/acpi/bbswitch <<< ON
# to turn it off
tee /proc/acpi/bbswitch <<< OFF

# to check for the error
dmesg | tail -l

# to auto-distable the card on boot
echo "blacklist nouveau" >> /etc/modprobe.d/blacklist
echo "bbswitch load_state=0" >> /etc/modules
update-initramfs -u

Wireless

download the driver from from: http://www.broadcom.com/support/802.11/linux_sta.php and read the instruction if you need more details: http://www.broadcom.com/docs/linux_sta/README.txt

tar -xvcf *.tar.gz
make

To install drive manually (and then lose it after the reboot):

sudo rmmod wl
sudo insmod /lib/modules/3.2.0-4-amd64/kernel/net/wireless/cfg80211.ko
sudo insmod /lib/modules/3.2.0-4-amd64/kernel/net/wireless/wl.ko
sudo modprobe wl

To make the change permanent

sudo sh -c 'echo -e "lib80211\ncfg80211\nwl" >> /etc/modules'
sudo update-initramfs -u

The power saving option for wireless may cause problem. I was getting a bunch of errors on my syslog:

crunchbang kernel: [  130.973790] ERROR @wl_cfg80211_get_station : Wrong Mac address

To temporarily fix the problem, you can do (First check your device with iwconfig and replace wlan0 appropriately):

sudo iwconfig wlan0 power off

To make the change permanent, the suggested edit is:

sudo sh -c 'echo -e "HOOK_BLACKLIST="wireless"' >> /etc/pm/config.d/blacklist
