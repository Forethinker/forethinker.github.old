Setting up latex

March 17, 2014 [ hack ] #latex Read in 1 min.

The OS I am using is Windows 7

Install MikTex and TexMaker in that order.

If you get a weird error like 'Could not start the command', just reset the settings under:

Options-->Settings file-->Reset settings

or re-install the program. I had to do this once before.

MikTex has it own executable for updating its packages. In case you have installed MikTex for all users on your machine, just make sure to use mitex-update.exe instead of miktex-udpate_admin.exe. If you have not run it for a long time, it will first attempt to remove the packages. Apparently (as of 2.9), you cannot remove and install at the same time, so you have to run the update twice.
