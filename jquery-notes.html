jQuery Notes

July 27, 2013 [ web ] #javascript Read in 7 min.

Try jQuery

jQuery Video Tutorial by Derek Banas @ newthinktank.com

The Basic

Get the latest jQuery downloaded:

<script type="text/javascript" src="http://code.jquery.com/jquery-latest.js"></script>

Or Google's version for faster download speed:

<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.0/jquery.min.js">

jQuery manipulates the DOM, tree-like representation of HTML file. It also handles the difference of JavaScript engine in different browsers. The following code waits for the document to finish loading, this is often always used for jQuery:

jQuery(document).ready(function(){
    <code>
});

jQuery objects are created by surrounding the HTML element (like CSS) in $(...). They are manipulated by jQuery functions.

  • text('newContnet'): changes the content between elements. The argument can be a string or a number
  • attr('attribute', 'value'): returns the value of the HTML element attribute, or change the value if the second argument is specified.
  • .css('property', 'value'): multiple values can be specified with surrounding curly braces and comma as the separate, and colon as the specifier. (i.e. JS object) If only property is specified as a argument, the value is returned. This function should be used sparingly as it is a better practice to store all CSS styles in the stylesheet. toggleClass() is recommended when possible.

jQuery selector:

  • $("div:has(h4)"): gets all div's that has h4 element
  • $("div[id]"): gets all div's that has id attribute
  • $("li[class='hero']"): matches a specific attribute
  • $("li[class^='h']"): matches a attribute with a partial match in the beginning
  • $("li[class$='h']"): matches a attribute with a partial match in the end
  • $("li:contains('Richards')"): contains searches through the content (val) of the element specified.
  • $("#numbers tr:odd"): this is just an example of selecting all odd rows that are decedents of #numbers
  • element:first - means is this element the first one. There are other pseudo selectors like :last, :odd, :even, :first-child, :last-child, :nth-child(2), :eq(2), :gt(0), and :lt(3). The index starts at 0, so :even selects the first one.

Traversing

  • find(): finds the decedent faster than css-style selector
  • filter(): use another element for selection, faster than , css-style selector
  • first(): finds the first element --
  • last(): finds the last element --
  • next(): traverse to the next sibling
  • prev(): traverse to the previous sibling
  • parent(): only looks at the immediate ancestor.
  • parents(): looks at all ancestors, and returns all matches.
  • closest(): looks at all ancestors, as well as the originating element, and returns the first match.
  • closest(): traverse up in the hierarchy and return 0 or 1 matching node
  • children(): direct children

Modifying an element in DOM

First create a node: $var price = $('<p>From $399.99</p>');, then use the variable to refer what to add. It is a convention to start the variable with $ if it is a jQuery node. If the node is not created as a variable, then just the string part (enclosed in quote) can be directly placed as a function argument. The element to append may be specified with a selecton, in which case it is moved to the specified anchor node.

$vacation = $('.vacation').append(price);
  • append(): new node is placed as the last child of the anchor node
  • prepend(): -- first --
  • after(): new node is placed after the anchor node as a sibling
  • before(): -- before --
  • remove(): removes the selection (elements) and its contents
  • empty(): removes all the contents of the selection
  • hasClass(): checks for the existence of the specified class on the element
  • addClass('className'): adds class
  • toggleClass('clasName'): toggles class
  • removeClass('className'): removes class

appendTo(), preprendTo(), insertBefore(), insertAfter() reverse the syntactic placement of the new node and the anchor node.

price.appendTo($('.vacation'));

jQuery object

this is not a jQuery object. $(this) is. Here is an example of creating an user-specified jQuery object

var price = $('<p>From $'+amount+'</p>');

Refactoring and reuse of jQuery objects can be done by storing it in variable. This is also recommended.

on, off, and namespace

on(<event[.namespace]]>, <target>, <event handler>): combines bind() + delegate() + bind() and used to specify events to be watched and handlers to invoke. Specifying <target> is called, "Event delegation". The event handlers do not have () at the end of the function name because then the function would be executed immediately when the jQuery script is loaded.

Events can be specified with namespace, so that they could be referred specifically with off() later, which is a newer version of 'unbind()'. Namepsace also works with trigger(). 

$('button').on('click.prev', prev);
$('button').on('click.next', next)
$('button').off('click.prev', prev)
$('button').on('.next', next)

Types of Events:

  • Forms: blur, change, focus, focusin, focusout, select, submit
    • focus+ bubbles, focus does not
  • Mouse: click, dblclick, focusout, hover, mousedown, mouseenter, mouseleave, mousemove, mouseout, mouseout, mouseup, toggle
  • Keyboard: focusout, keydown, keypress, keyup
    • keypress returns uppercase of the character pressed (e.g a => '97'). keydown and keypress returns the default value (e.g. a => '65')

All events can be used as function. They are syntactic sugars that actually use on() underneath.

Trigger

trigger('event' [,extraArguments]): try to mimic the event as if the user-triggered it. This provide an alternative trigger point where the events can be linked together. (e.g. show-all button to each photo). See plugin in AJAX for an example. A customer event can be created with extra arguments, which is evaluated at the time the event is triggered

$('#foo').on('custom', function(event, param1, param2) {  
    alert(param1 + "\n" + param2);
});
$('#foo').trigger('custom', ['PassedToParam1', 'PassedToParam2']);

triggerHandler('eventType', [,extraArguments]): execute all handlers attached to an element for an event does not cause the default behavior, only affects the first matched element, events are not bubbled up

Value

data-element-name='value' in html can be looked up with data('tag-name') in jQuery and changed with data('tag-name', value) The value returned is a string, and it can be converted to a number with + prepended to the $() query.

example of using val(), is(), fadeIn(), fadeout(), fadeToggle(), hide(), show() :

$('#bringItBack').click(function()) {
    if ($('#clickToHide').is(':visible')) {
        <!--$('#clickToHide').hide();-->
        $('#clickToHide').fadeOut(2000)
        $(this).val('Bring Back');
    }
    else
    {
        <!--$('#clickToHide').show();-->
        $('#clickToHide').fadeIn();
        $(this).val('Delete Text')
    }
});

There are more transition functions like slideUp(), slideDown(), slideToggle(). They all act on display CSS property. The animation speed can be adjusted in the second argument with number (ms) or keywords like: 'fast' (200) 'slow' (600). It is 400ms by default.

animate() is used like css(), but make the transition to look animated. Just like toggleClass() is recommend for css(), use of CSS3's transition property is recommend along with toggleClass()

Event

event is a variable that gets passed to any event handler, just specify it in the argument

Any event of jQuery is propagated up to the root of DOM tree. In order to prevent the parents to be triggered by this bubble, stopPropagation() function is used. And in order to prevent some default actions or event listeners from being triggered the preventDefault() function is used to stop it. an example is a jQuery button that has href='#' attribute. When the bubble reaches the DOM root, it triggers the default behavior which would bring the page back to top.

  • e.target.nodeName: contains the name of the element triggered by the latest event
  • e.timeStamp: contains the time the latest event was triggered
  • e.screenX: x-coordinate of the mouse
  • e.screenY: y-coordinate of the mouse
  • e.keyCode: key in the corresponding ASCII decimal. It can be converted String.fromCharCode(e.keyCode)

Debugging:

  • alert(): display the message in a pop-up box
  • .length: returns the number of elements matching the specified jQuery object

Other

  • text() vs html(): html string could contain the elements which will be displayed according to the HTML spec, while the text will only show the contents between the elements specified.

'window' can be looked up with jQuery

$(window).resize(resizedWindow);

function resizeWindow()
{
    $("#second").html("Window was resized W: " + $(window).width() + "H: " + $(window).height());
}
