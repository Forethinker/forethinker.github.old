Setting up a filter on your (unrooted) phone

July 12, 2014 [ hack ] Read in 3 min.

Introduction

I do not want to root my phone because then I have to update my phone manually. And rooting my phone is the best way to kill my weekend time. I am going to have fun, but I am not going to get much done - I know because that is what I did with PDAs back in high school. So I was looking for a way to filter some Social Networking Services(SNS). I know that it is difficult to block the access completely, because at the end of the day, if I am determined, I can break anything, but some deterrences usually do the trick. I will add some apps on the bottom if you want to protect your filtering from getting disabled.

Journey

I tried several apps, and spent some time on Block It!, but not only you cannot trust this app, but it slows your phone and internet speed down unusably. I also tried AdBlock, you have to download APK because Google Play Store removed it due to security reasons, unfortunately, without rooting, you cannot setup a custom fitering. So here comes the solution FilterProxy.

FilterProxy to the rescue

The bland interface will probably raise a flag to an average user. You do need some patience to set it up. I am just going to write exact steps I took.

  • There is a ... (in vertical) at the right bottom corner. Then I press Export.
  • I copied it and saved in Dropbox.
  • I opened the file up on my computer and edit it with Vim. Vim-Surround is useful here to put '' tag around all the words. These words that you are going to block are regular expression statements. To put thing simply, if specify a word within that flag, it will block all URLs that has that word.
  • I copied my custom keywords: facebook, instagram, pinterest

  • I also copied some other keywords to mimic AdBlock's functionality. The credit goes to Baron Schwartz, the author of High Performance MySQL

Changing your Network to use the FilterProxy

Please refer to the official manual for more details.

WIFI

  • "WIFI"
  • Select your network and press it until you see a pop up
  • "Modify network"
  • "Show advanced options"
  • "Proxy", set "Manual"
  • In "Proxy hostname", set "localhost"
  • In "Port", set "8000"(Default port number of FilterProxy is 8000)

Mobile

  • In "Main-Setting" Tab, push "setting"
  • "Mobile networks" -> "Access Point Names" -> {Using Access Point}
  • In "Proxy", set "localhost"
  • In "Port", set "8000"(Default port number of FilterProxy is 8000)
